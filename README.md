### Description
This is a book flighting site with the 4 best places to visit of the year. 
Please note, this app is still a work in progress. Feel free to make adjustments as you best see fit.

### Installation

**Front End**
1. clone or download repo
2. Do `npm start` to start the app

**Backend: Server Side**
1. `npm install`
2. `nodemon sever.js` or `node server.js` to run the server listening at port 8080
